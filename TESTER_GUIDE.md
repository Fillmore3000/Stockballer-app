# StockBaller Beta Testing Guide

Welcome to the StockBaller beta test! This guide will help you get set up and start testing.

## Quick Start (5 minutes)

### Step 1: Install MetaMask

1. Go to [metamask.io](https://metamask.io/download/)
2. Install the browser extension (Chrome, Firefox, or Brave recommended)
3. Create a new wallet and **save your recovery phrase securely**

### Step 2: Add Base Sepolia Network

1. Open MetaMask and click the network dropdown (top left)
2. Click "Add Network" → "Add a network manually"
3. Enter these details:
   - **Network Name:** Base Sepolia
   - **RPC URL:** `https://sepolia.base.org`
   - **Chain ID:** `84532`
   - **Currency Symbol:** `ETH`
   - **Block Explorer:** `https://sepolia.basescan.org`
4. Click "Save"

### Step 3: Get Test ETH (for gas fees)

1. Go to [Alchemy Base Sepolia Faucet](https://www.alchemy.com/faucets/base-sepolia)
2. Enter your wallet address (copy from MetaMask)
3. Request test ETH (you only need a tiny amount for gas)

### Step 4: Open StockBaller

1. Go to the app URL (provided by the team)
2. Click "Connect Wallet" in the top right
3. Approve the connection in MetaMask

### Step 5: Get Test USDC

1. Once connected, click "Get Test USDC" on the home screen
2. Confirm the transaction in MetaMask
3. You'll receive $10,000 test mUSDC

## How to Test

### Buy Player Tokens

1. Go to the **Market** tab
2. Browse players or use filters (Top Gainers, Best Form, etc.)
3. Tap on a player to see details
4. Click "Buy" and enter an amount
5. Review the quote and confirm
6. Approve USDC spending in MetaMask (first time only)
7. Confirm the buy transaction in MetaMask

### Sell Player Tokens

1. Go to the **Portfolio** tab
2. Find the position you want to sell
3. Click "Sell" 
4. Enter quantity and confirm
5. Approve the transaction in MetaMask

### View Your Portfolio

1. Go to the **Portfolio** tab
2. See your total value, P&L, and individual positions
3. Pull down to refresh from blockchain

## FAQ

**Q: Is this real money?**
A: No! This is on Base Sepolia testnet. All tokens have no real value.

**Q: Why do I need MetaMask?**
A: MetaMask is your Web3 wallet that allows you to interact with the blockchain.

**Q: Why did my transaction fail?**
A: Common reasons:
- Not enough ETH for gas (get more from faucet)
- Transaction was rejected in MetaMask
- Network congestion (try again in a minute)

**Q: Can I test on mobile?**
A: Yes! Open in your mobile browser. MetaMask mobile app also works.
