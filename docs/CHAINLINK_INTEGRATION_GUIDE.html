2. What is Chainlink CRE?

CRE = Compute Runtime Environment

It's a framework that lets you write workflows that:

  • Trigger on a schedule (cron) or blockchain event
  • Fetch external data (APIs, databases)
  • Compute results (your price formula)
  • Write to blockchain with cryptographic proof
